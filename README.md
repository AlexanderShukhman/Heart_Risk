# Дашбоард собранный Perplexity.Pro

Cтатический JS-дашборд без бэкенда, который работает прямо с CSV-файлом и поддерживает интерактивные фильтры, графики и таблицу записей. По содержимому файла видны поля возраста, пола, типа боли в груди, артериального давления, холестерина, ИМТ, статуса курения, физической активности и целевого признака `heartattackrisk`, поэтому дашборд построен именно вокруг этих признаков. 

## Что внутри

В приложении есть фильтры по полу, типу боли, курению, активности, диапазону возраста, диапазону холестерина и бинарному признаку риска инфаркта. Также добавлены KPI-блоки, диаграмма риска по полу, распределение по типам боли, scatter-график возраста против холестерина и график риска по статусу курения, плюс таблица строк после фильтрации. 

## Как открыть

Откройте HTML-файл в браузере и при необходимости выберите CSV через поле загрузки; рядом уже приложен тот же набор данных, так что страница сможет использовать его сразу. Поскольку это полностью клиентское приложение, никакой сервер, база данных или API не нужны, а вся обработка идет локально в браузере. 

## Особенности данных

В CSV действительно присутствует целевой столбец `heartattackrisk` со значениями 0 и 1, поэтому можно напрямую показывать долю наблюдений с высоким риском и сравнивать группы между собой. В файле также встречаются категориальные признаки вроде `gender`, `chest pain type`, `smoking status` и `physical activity`, что хорошо подходит для интерактивной сегментации на дашборде.
