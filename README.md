# Дашбоард собранный Perplexity.Pro

Cтатический JS-дашборд без бэкенда, который работает прямо с CSV-файлом и поддерживает интерактивные фильтры, графики и таблицу записей. По содержимому файла видны поля возраста, пола, типа боли в груди, артериального давления, холестерина, ИМТ, статуса курения, физической активности и целевого признака `heartattackrisk`, поэтому дашборд построен именно вокруг этих признаков.
